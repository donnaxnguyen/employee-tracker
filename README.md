# Employee Tracker
This is an command-land application that allows a user to organize their employee database!

## Delivery
With this application, a user can:
  * Add employees, departments, and roles
  * View employees, departments, and roles


## How it's Made
  * InquirerJs
  * Console.table
  * MySQL
